# Uploading extensions
Even tho this is my extension gallery you are free to upload your own extensions!
# First steps
First of all you have to code your extension, there are some documentations by penguinmod and turbowarp on how to do it
After your extension is done you have to go to the folder Extensions
and upload your extension
then make a pull request
# Banners and Json
After making the pull request go to your fork and draw the banner for your extension
after that you can upload it oj the folder Banners
Everything is almost done now you have to go to code
then go to Extensions.json
if you know json you should know how to edit it already
change the author name to your name
and the link to your scratch or github account
on the extension like change the link to the name of your extension (make sure to begin with Extensions/) before writing the name
Then do the same with the banner, change it to the path of your banner (make sure to begin with Banners/)
When you are done just wait if i accept it!

# Rule 1
No AI generated extensions
# Rule 2
extensions made with turbobuilder or Extforge are accepted BUT they are more likely to not be accepted and if accepted they will have a tag of possible low quality
# Rule 3
Joke extensions are accepted BUT they need to have a use and not be just 1 block with no use
# Rule 4
Please check for extensions that already exist and avoid to replicate them
# Rule 5 
if you want to get your extension accepted please also make it SFW
